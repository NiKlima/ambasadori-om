# OM Brand Tokens — для Claude Design

Извлечено из официального Brand Identity PDF (49 стр.). Используй вместо PDF.

## Бренд

- **Название**: OM (вода, Молдова) · sub-brand для нас → **OM Activ**
- **Слоган**: «OM este decizia mea» (рум.: «OM — моё решение»)
- **Тон**: спокойная уверенность, личное решение, движение, осознанность

## Палитра (главная, ~80% макета)

| Token            | HEX       | RGB             | Использование                            |
|------------------|-----------|-----------------|------------------------------------------|
| `--om-blue`      | `#0047b9` | 0 71 185        | Доминанта, ~50% площади                  |
| `--om-white`     | `#ffffff` | 255 255 255     | Фон, ~25%                                |
| `--om-ink`       | `#231f20` | 35 31 32        | Текст, акценты, ~15%                     |
| `--om-magenta`   | `#d6006d` | 214 0 109       | Акцент CTA, маркер новизны, ~10% максимум|

Допвцвета (cream/sand из текущего globals.css) — оставляем как тёплый
оттенок белого для длинных текстовых блоков, но в новом макете blue/white/
ink/magenta доминируют.

## Типографика

- **Семейство**: **TT Firs Neue** (Bold + Regular)
  - Geometric modern-grotesque
  - Поддерживает Latin + Cyrillic
  - Платный (TypeType). На вебе можно заменить на **Inter** (близкий
    grotesk, бесплатный, уже подключён в проекте) или подключить TT Firs
    Neue через лицензию TypeType.
- **Иерархия**: жирные заголовки (700/800), regular для body. Никаких
  тонких/light весов.

## Лого

- **OM wordmark** — буква «M» имеет характерный угловой срез по центру
  (см. `02-visual-vocabulary.png` и `03-om-activ-logo.png`).
- **OM Activ logo** — наш sub-brand для трейнеров. Три варианта: main,
  with line, additional (подпись «activ» под OM).
- **OM Infinite Line** — фирменный графический элемент: горизонтальные
  синие полосы, отходящие от лого вправо. Это визуальная подпись бренда.

## Запреты (incorrect usage)

- Никаких обводок/strokes на лого
- Никаких теней / shadow / glow
- Не искажать пропорции
- Не накладывать паттерны / текстуры внутрь букв
- Не использовать неутверждённые цвета
- Не размещать на «грязных» / пёстрых фонах

## Применение к нашему UI (рекомендации)

### CSS-переменные (заменить в `src/app/globals.css`)
```css
--om-blue: #0047b9;          /* было #1e7ad9 — заменяем на бренд */
--om-blue-dark: #003a96;     /* hover-состояние */
--om-ink: #231f20;
--om-cream: #ffffff;         /* фон → чистый белый, не cream */
--om-sand: #f4f4f5;          /* нейтральный серый для секций */
--om-muted: #6b6b6e;
--om-magenta: #d6006d;       /* НОВЫЙ — акцентный CTA, badges */
```

### Где применить magenta (с осторожностью)
- Бейдж «новый челлендж» / «осталось 3 дня»
- Точка «вы здесь» в leaderboard
- Иконка «Поделиться» при ховере
- Стрелка CTA «Стать амбассадором»

### Где применить «infinite line»
- Hero-секция: горизонтальные синие полосы за заголовком
- Разделители между секциями лендинга
- Декоратор в TrainerModal под именем тренера

### Радиусы
- Брендбук показывает прямоугольные блоки и round-логотип «OM».
- В UI: оставляем `rounded-3xl` для карточек (мягкость), но кнопки
  делаем pill-shape (`rounded-full`) — соответствует логотипу.

### Тени
- Брендбук — никаких теней. В вебе разрешаем мягкие `shadow-sm` для
  карточек на белом фоне, но без glow / drop-shadow / colored shadows.
